Coordinates Loop Route

How to build a route:
  • The route path connects each coordinate you add point-by-point.
  • Set the Point Type: Origin, Waypoint (direction change), Road Segment (road continuation), Alight or Board Stop, or Destination.
  • There can only be one Origin and one Destination.
  • If the Destination coordinates match the Origin, it is marked as a loop.
  • Waypoints, Road Segments, and Stops are automatically numbered sequentially.
